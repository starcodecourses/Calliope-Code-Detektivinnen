# Calliope-Code-Detektivinnen
Dieser Kurs bringt Mädchen die Welt das Programmieren nahe, indem sie durch eine grafische Programmieroberfläche (Makecode) verschiedene Detektiv-Gadgets mithilfe des Calliope mini Boards erschaffen. 

Die Programmierumgebung ist der MAKECODE Editor auf der [calliope.cc Seite] (https://calliope.cc/programmieren/editoren).

Der Kurs ist folgendermaßen aufgebaut:
Alle einzeln:
1. Detektivinnen-Badge
2. Thermometer, Kompass, Magnetfelddetektor

Dann teilen wir die Mädchen auf 3 Gruppen auf, die je das Fangspiel, Morse oder den Lügendetektor bearbeiten.

Wenn danach Zeit übrig bleibt, können die Teilnehmerinnen entweder noch eine andere der beiden bisher nicht bearbeiteten Aufgaben lösen oder sich selbstständig mit dem Board beschäftigen.
